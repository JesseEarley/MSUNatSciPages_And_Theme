About NatSci

This is an example of a 2 column Section Front page, with a 4 column footer. Side navigation shows up to 3 levels deep. Header represents a college header, note the abscence of a department name, only the college name is present. Please see HTML comments for more information.

As a Section Front, there should be a page or section setting to change the primary navigation to only display child pages within that section, rather than the sites overall primary navigation. This option allows for the Section Front page to act as a "homepage" of a mini/micro-site within a parent site.
